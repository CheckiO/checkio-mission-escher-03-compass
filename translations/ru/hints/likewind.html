for i in range(len(seaside)):

Сперва вы создаете переменные со списками координат и присваиваете им значения [0, 0]. После чего с помощью цикла проверяете каждый элемент двумерного массива и записываете его координаты в соответствующую переменную.
Как, зная координаты, мне посчитать общее количество шагов?
Вам необходимо определить разницу координат между вашим местоположением и положением одной вещи, а затем выбрать максимальное из двух значений - разницы по-горизонтали и по-вертикали. Это можно сделать с помощью функции max(). После чего вам нужно будет выполнить аналогичные действия по отношению к двум другим предметам и посчитать сумму трех расстояний.

Для удобства, вы можете описать данную часть решения в отдельной функции: 
def dist(a, b):
    return max(abs(a[0]-b[0]), abs(a[1]-b[1]))
где a и b - списки с координатами вашей позиции и позиции одного из предметов.
