一日にも満たない短い旅だから問題は何もないはずでした。 しかし水平線上に島が見えてきた頃に恐るべき嵐がやってきました。 あなたと船員たちが体験したことを言葉にするのは難しいほどです。 実際に島にたどり着いた人々の中であなたは唯一の生存者になりました。 荒波は双方の船を破壊し残っていた船底は最後は崖によって粉々になりました。 他の船員の誰かがなんとか生き延びているかもしれませんがそれも望み薄です。 今あなたには二つの仕事があります。 城からハイパーキューブを取ってくること。そして呪われた島から脱出する方法を見つけること。

一つずつ進めましょう。 手始めに地形を把握するためにコンパスと地図と望遠鏡が必要です。 急いで！ 自然は予測不可能です。 もしこれらを時間内に入手できなければあなたは砂浜に埋もれるか海の藻屑となるでしょう。

あなたの仕事は今いる場所から各々の品物を入手するのに必要なステップ数の合計を数えることです。('C' - コンパス), ('M' - 地図), ('S' - 望遠鏡) 今あなたがいる場所は 'Y' です。 Y から C, M, S の各々への距離の合計であり、全ての品物を得るための最短距離( 例えば Y -> C -> M -> S) ではありません。
あなたは 8方向に歩けるのに注意して下さい。- 左, 右, 上, 下 そして斜め (いづれの斜め方向でも可能)

入力: あなたと 3つの品物の位置をあらわした配列

出力: あなたから 3つの品物への距離の合計
