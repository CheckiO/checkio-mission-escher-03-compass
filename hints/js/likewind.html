Could you tell me how to start solving this mission?
First of all, you need to determine your own coordinates, as well as the coordinates of C (compass), S (spyglass), and M (map).
How can I do that?
First, you create variables with the lists of coordinates and assign them values [0, 0]. After that, using a loop, you need to check each element of a two-dimensional array and write its coordinates in the corresponding variable.
